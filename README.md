## 📁 Projetos

### 🏨 01 — Sistema de Gerenciamento de Hotel
> **Semana 1 · Concluído**

Sistema de console para gerenciar quartos e hóspedes de um hotel.

**Conceitos aplicados:**
- Orientação a Objetos (Classes, Objetos, Encapsulamento)
- Associação entre classes (`Quarto` ↔ `Hospede`)
- Arrays de objetos
- Construtores e métodos
- Menu interativo com `Scanner`, `while` e `switch`

**Como executar:**
```bash
# Clone o repositório
git clone https://github.com/seu-usuario/jornada-java.git

# Abra o projeto no IntelliJ IDEA
# Execute a classe Main.java dentro de hotel-sistema/
```

**Estrutura do projeto:**
```
hotel-sistema/
└── src/
    └── br/com/hotel/dominio/
        ├── Hospede.java
        ├── Quarto.java
        ├── Hotel.java
        └── Main.java
```
